# vtherm_adaptive_tpi

Adaptive TPI plugin scaffold for Versatile Thermostat.

This repository contains the initial project structure for an external
`adaptive_tpi` proportional algorithm based on `vtherm_api`.

Current status:

- Home Assistant integration scaffold
- external proportional algorithm factory registration
- minimal runtime-safe handler
- initial `adaptive_tpi/` module split for future implementation

The adaptive control logic is not implemented yet.
